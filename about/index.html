# 开源项目

# 乐办公：

  • 项目描述：本项目属于个人项目，采用前后端分离的开发模式，目的是为了提高中小型公司的管理效率，使得对员工、部门、薪资、职位等管理变得更加简单更加高效。针对不同的用户登录，分配不同的权限菜单，更加规范的进行管理。另外还包括当新员工入职时，获取员工的信息以及邮箱完成自动发送入职欢迎邮件的功能。
  • 项目架构：SpringBoot+SpringSecurity+MyBatis-Plus+Redis+RabbitMQ+ElementUI+Vue
  • 技术要点：
    • SpringSecurity整合JWT登录，配置登录授权过滤器
    • Redis对不同权限能够访问的菜单进行缓存
    • 使用RabbitMQ+SpingBoot-Mail实现发送邮件任务
    • EasyPoi实现员工的批量导入导出
  • 项目地址：https://github.com/yy03/Leban (opens new window)
