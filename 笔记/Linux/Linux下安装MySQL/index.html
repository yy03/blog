Linux操作系统安装MySQL

2020-11-19
笔记
Linux

# 1. MySQL的安装方式

  • 二进制安装
    • RPM版本或yum安装：安装卸载简单，可定制性差
    • glibc版本：可定制性相比rpm包灵活些，需要手动初始化数据库
  • 源码安装；可定制性强，需要手动初始化数据库

# 2. MySQL的glibc版本安装

1. 安装需求：

选项 值
安装路径： /usr/local/mysql
数据路径： /usr/local/mysql/data
端口号： 3306

2. 安装（查看官方说明）

MySQL官方安装教程 (opens new window)

  • 创建MySQL用户

    MySQL软件比较特殊，在MySQL启动和运行过程中，都需要一个特定的用户，即MySQL用户。所以在Linux操作系统中要创建一个数据库专用用户——MySQL用户，让该用户所属组也为MySQL。

    [root@localhost mysql]# useradd -r -s /sbin/nologin mysql
[root@localhost mysql]# id mysql

  • 清空系统中原有的mariadb的配置文件（/etc/my.cnf）

    [root@localhost /]# rm -f /etc/my.cnf

  • 在MySQL工作目录中创建mysql-files文件夹，并修改权限

    [root@localhost mysql]# mkdir mysql-files
[root@localhost mysql]# chown mysql:mysql mysql-files
[root@localhost mysql]# chmod 750 mysql-files
#说明：mysql-files目录为secure_file_priv系统变量提供一个位置，该位置将导入和导出操作限制到特定目录

  • 初始化数据库（重要），会产生一个随机密码

    [root@localhost mysql]# bin/mysqld --initialize --user=mysql

    如果初始化成功，会在mysql目录下生成一个data文件夹，且文件夹中至少要有一个mysql文件夹

  • 设置安全加密连接

    [root@localhost mysql]# bin/mysql_ssl_rsa_setup

  • 启动数据库

    [root@localhost support-files]# cp mysql.server /etc/init.d/mysql
[root@localhost support-files]# service mysql start
Starting MySQL.Logging to '/usr/local/mysql/data/localhost.localdomain.err'.
 SUCCESS!

    mysql日志存放目录'/usr/local/mysql/data/localhost.localdomain.err'

  • 将mysql设置开机启动

    [root@localhost /]# chkconfig --add mysql
[root@localhost /]# chkconfig --list

3. 后续配置

  • 更改数据库管理员密码

    [root@localhost mysql]# bin/mysql -uroot -p

mysql> set password = '123456'
    -> ;

mysql> flush privileges;
Query OK, 0 rows affected (0.01 sec)

  • 将mysql添加到环境变量

    [root@localhost /]# echo 'export PATH=$PATH:/usr/local/mysql/bin' >> /etc/profile
[root@localhost /]# source /etc/profile

  • 手工定义MySQL的配置文件（非常重要）

    vim /usr/local/mysql/my.cnf

[mysqld]
basedir=/usr/lcoal/mysql
datadir=/usr/local/mysql/data
socket=/tmp/mysql.sock

选项说明：
[mysqld]				代表针对服务器进行配置
basedir					代表数据库的安装目录
datadir					代表数据目录，以后专门用于存放数据文件的（核心目录）
socket					代表套接字文件，专门为客户端与服务器端连接提供一个桥梁

service mysql restart
